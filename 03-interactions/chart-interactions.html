MY FINAL ASSIGNEMENT

Some examples that show how to use the techniques from intro-interactions.html in a D3 data visualization.

Bar chart

04-full-bar-with-hover.js

Hover over each bar to see the number of asylum seekers per country. The bar also changes color when hovered over.

The annotation comes from a new field in the data that's been created called Comma Value, which uses regex inserts commas into the number value.

Countries with the highest numbers of asylum seekers to the United States in 2024

Source: United Nations High Commissioner for Refugees

Scatterplot

05-scatterplot-interactions.js

Each circle is enlarged and returns a country name when hovered over. The circles also turn orange when you click on them.
