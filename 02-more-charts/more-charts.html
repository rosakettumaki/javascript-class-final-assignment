More chart types

More standard chart types to show how SVG shapes are used to make data visualizations.

Bar chart

01-bar-chart.js

Drawing bars isn't super straightforward (like figuring out the height), since SVG coordinates start from the top-left.

There's also some data subsetting going on in the beginning of the script, which I'd generally recommend against -- it's often more efficient to load in a smaller dataset that's already been cleaned up and subset in Python/R/Excel/etc -- but these array functions could come in handy in other contexts!
